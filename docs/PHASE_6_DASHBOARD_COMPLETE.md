### 3. 圓形進度圖（卡路里）

**視覺設計**:
- 使用 SVG `<circle>` 元素
- 背景圓（灰色）+ 進度圓（彩色）
- 中心顯示數字和百分比
- 旋轉 -90° 從頂部開始

**動態顏色**:
- 🟢 綠色: 90-110% 達成率（理想範圍）
- 🟠 橙色: >110% 超過目標
- 🔵 藍色: <90% 未達目標

**技術實作**:
```typescript
// SVG 圓形進度
<circle
  cx="96"
  cy="96"
  r="88"
  strokeDasharray={`${2 * Math.PI * 88}`}
  strokeDashoffset={`${2 * Math.PI * 88 * (1 - progress / 100)}`}
  className={getProgressColor(current, goal)}
/>
```

**數學原理**:
- 圓周長: `2πr = 2 × π × 88 ≈ 553`
- 進度 60%: offset = `553 × (1 - 0.6) = 221`
- strokeDasharray 定義虛線長度
- strokeDashoffset 控制起點偏移

**視覺效果**:
```
        100%
         ↓
    ┌────────┐
    │  1850  │
90% │        │ 10%
←───│  kcal  │───→
    │        │
    │  92%   │
    └────────┘
         ↑
        0%
```

## 📊 數據計算邏輯

### 進度百分比
```typescript
const getProgress = (current: number, goal: number) => {
  if (!goal) return 0;
  return Math.min((current / goal) * 100, 100);
};
```

### 週平均
```typescript
const weeklyAverage = weeklyData.reduce((sum, day) => 
  sum + day.calories, 0
) / weeklyData.length;
```

### 長條圖高度
```typescript
const maxCalories = Math.max(
  ...weeklyData.map(d => d.calories),
  goals?.dailyCalorieGoal || 2000
);

const barHeight = (day.calories / maxCalories) * 100;
```
